About This AI Intake System

What it does

The system collects symptom information, generates follow-up questions, and creates a structured summary that a user could share with a healthcare professional.

What it does not do

It does not diagnose, prescribe medicine, identify diseases, or replace professional medical judgement.

AI Use

The app can use an OpenAI-compatible model provider such as APIFree to generate follow-up questions and summaries. If no API key is configured, deterministic fallback logic keeps the demo reproducible.
